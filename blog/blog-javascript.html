JavaScript Basics

March 2026 • JavaScript • 20 min read

← Home All Posts

JavaScript runs the web. Every interactive site uses it. This guide covers the fundamentals you need to build modern web applications.

Variables

let name = "cjboon";
const PI = 3.14159;
var oldWay = "don't use var";

let = can change. const = cannot change. Avoid var.

Data Types

let text = "hello";      // string
let num = 42;            // number
let bool = true;         // boolean
let arr = [1, 2, 3];    // array
let obj = {a: 1};       // object

Functions

// Function declaration
function greet(name) {
    return "Hello, " + name;
}

// Arrow function
const add = (a, b) => a + b;

Conditionals

if (age >= 18) {
    console.log("adult");
} else {
    console.log("minor");
}

Loops

// For loop
for (let i = 0; i < 5; i++) {
    console.log(i);
}

// For each
arr.forEach(item => console.log(item));

DOM Manipulation

// Select element
const btn = document.querySelector('#myButton');

// Add event
btn.addEventListener('click', () => {
    alert('Clicked!');
});

// Change content
document.getElementById('output').textContent = 'New text';

Async JavaScript

// Fetch API
fetch('https://api.example.com/data')
    .then(res => res.json())
    .then(data => console.log(data))
    .catch(err => console.error(err));

Modern Async: async/await

async function getData() {
    try {
        const res = await fetch('https://api.example.com/data');
        const data = await res.json();
        return data;
    } catch (err) {
        console.error(err);
    }
}

You're ready to start building!

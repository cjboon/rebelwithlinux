Setting Up Your First VPS

March 8, 2026 • Server Administration • 15 min read

← Home All Posts

So you've just rented your first Virtual Private Server (VPS) running Debian Linux. Congratulations! But now what? This comprehensive guide walks you through securing and setting up your server from scratch.

First Steps After Login

Update Your System

sudo apt update
sudo apt upgrade -y

Create a New User

adduser yourname
usermod -aG sudo yourname

SSH Security

Key-Based Authentication

ssh-keygen -t ed25519
ssh-copy-id yourname@your-server-ip

Harden SSH Config

sudo nano /etc/ssh/sshd_config

Change these settings:

  • PasswordAuthentication no
  • PermitRootLogin no
  • PubkeyAuthentication yes

Enable UFW Firewall

sudo ufw allow OpenSSH
sudo ufw enable
sudo ufw status

Essential Software

Install Basic Tools

sudo apt install curl wget git vim ufw fail2ban

Fail2ban Protection

sudo systemctl enable fail2ban
sudo systemctl start fail2ban

Setting Up a Web Server

Install Nginx

sudo apt install nginx
sudo systemctl enable nginx
sudo systemctl start nginx

Firewall for Web

sudo ufw allow 'Nginx Full'

Automated Backups

Set up automated backups using restic or borgbackup. Check out our Server Backups with Restic guide.

Your VPS is now secure and ready for use! Remember to keep your system updated and monitor logs regularly.

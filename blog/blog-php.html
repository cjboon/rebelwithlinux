PHP Basics

March 2026 • PHP • 15 min read

PHP powers 75% of websites. WordPress, Drupal, Laravel—it's everywhere. Here's how to build dynamic web apps with PHP.

Why PHP?

PHP was built specifically for the web. It runs on the server, generates HTML, and sends it to the browser. Simple. Fast. Everywhere.

Installing PHP

sudo apt install php php-mysql php-curl php-gd php-mbstring php-xml

Your First PHP Script

<?php
echo "Hello, World!";
?>

Save as index.php and access via your web server.

Variables

$name = "cjboon";
$age = 25;
$is_active = true;

Arrays

$fruits = ["apple", "banana", "orange"];
echo $fruits[0];  // apple

// Associative array
$user = ["name" => "cjboon", "email" => "test@test.com"];
echo $user["name"];  // cjboon

Conditionals

if ($age >= 18) {
    echo "Adult";
} else {
    echo "Minor";
}

Loops

foreach ($fruits as $fruit) {
    echo $fruit . "\n";
}

Functions

function greet($name) {
    return "Hello, " . $name;
}

echo greet("cjboon");

Superglobals

PHP has built-in variables accessible anywhere:

// GET parameters
echo $_GET["name"];

// POST data
echo $_POST["email"];

// Session
$_SESSION["user"] = $user;

Connecting to MySQL

$pdo = new PDO(
    "mysql:host=localhost;dbname=mydb",
    "username",
    "password"
);

$stmt = $pdo->query("SELECT * FROM users");
while ($row = $stmt->fetch()) {
    echo $row["name"];
}

You're ready to build!

Essential Networking Commands

  • ip - Show/manipulate network interfaces and routing
  • ping - Test connectivity to a host
  • curl - Transfer data from/to a server
  • ss - Investigate sockets (modern replacement for netstat)
  • netstat - Network statistics (deprecated, use ss)
  • traceroute - Trace the path packets take to a host
  • dig - DNS lookup utility
  • nslookup - Query DNS servers
  • host - DNS lookup tool
  • arp - Show/manipulate ARP cache
  • route - Show/manipulate IP routing table
  • iptables - Firewall rule management
  • firewalld - Dynamic firewall manager
  • nmap - Network discovery and security auditing
  • tcpdump - Packet capture and analysis

How to play:

Regex basics:

  • \d - Match any digit (0-9)
  • \w - Match any word character
  • [aeiou] - Match any vowel
  • ^start - Match words starting with "start"
  • end$ - Match words ending with "end"
  • . - Match any single character
  • * - Match 0 or more
  • + - Match 1 or more

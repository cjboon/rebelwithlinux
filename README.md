## Why Free Software?

Because proprietary software often comes with baggage—telemetry, licensing fees, and terms of service that basically say "we watch you." FOSS puts you in control. You can see what the code does, modify it, and share it.

No judgment if you're coming from Windows or macOS. We've all been there. The command line is friendly once it clicks.

Setting Up Sign-In Functionality on a New Server
 =================================================

The login/registration system on this site requires:
1. PHP (server-side language to run api/auth.php)
2. MariaDB (database for storing user credentials)

Steps to set up:

1. INSTALL MARIADB
   - On Debian/Ubuntu: sudo apt install mariadb-server
   - Start the service: sudo systemctl start mariadb
   - Enable on boot: sudo systemctl enable mariadb

2. INSTALL PHP
   - On Debian/Ubuntu: sudo apt install php php-mysql
   - Ensure PHP can connect to MariaDB via socket

3. CONFIGURE DATABASE
   - Log into MariaDB: sudo mariadb -u root
   - Create database: CREATE DATABASE learning_platform;
   - Create user: CREATE USER 'webuser'@'localhost' IDENTIFIED BY 'webpass';
   - Grant permissions: GRANT ALL PRIVILEGES ON learning_platform.* TO 'webuser'@'localhost';
   - Create users table:
     USE learning_platform;
     CREATE TABLE users (
       id INT AUTO_INCREMENT PRIMARY KEY,
       email VARCHAR(255) UNIQUE NOT NULL,
       password VARCHAR(255) NOT NULL,
       created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
     );
   - Flush privileges: FLUSH PRIVILEGES;
   - Exit: exit

4. ENSURE MYSQL SOCKET EXISTS
   - The socket should be at: /run/mysqld/mysqld.sock
   - If not, check that MariaDB is running and the socket path is correct

5. WEB SERVER CONFIGURATION
   - Make sure your web server (Apache/Nginx) has PHP enabled
   - The api/ directory must be accessible and executable

WHAT WAS RUN ON THIS SERVER (2026-02-28):
================================================================================

# Update package lists
sudo apt update

# Install MariaDB and PHP
sudo apt install -y mariadb-server php php-mysql

# Start and enable MariaDB
sudo systemctl start mariadb
sudo systemctl enable mariadb

# Create database and user
sudo mariadb -u root -e "CREATE DATABASE IF NOT EXISTS learning_platform; CREATE USER IF NOT EXISTS 'webuser'@'localhost' IDENTIFIED BY 'webpass'; GRANT ALL PRIVILEGES ON learning_platform.* TO 'webuser'@'localhost'; FLUSH PRIVILEGES;"

# Create users table
sudo mariadb -u root -e "USE learning_platform; CREATE TABLE IF NOT EXISTS users (id INT AUTO_INCREMENT PRIMARY KEY, email VARCHAR(255) UNIQUE NOT NULL, password VARCHAR(255) NOT NULL, created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP);"

# Verified:
# - Socket exists at /run/mysqld/mysqld.sock
# - PHP can connect to MariaDB successfully

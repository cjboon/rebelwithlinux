// Linux SysAdmin Pathway

WHAT IS LINUX SYSADMIN?

Linux system administration is managing Linux servers—the computers that run the internet. Web servers, databases, email servers, DNS, file storage. When you host something, a Linux server is what makes it available. Sysadmins keep these systems running, secure, and performant.

WHY LINUX?

Linux runs most of the world's servers. It's free, open source, and doesn't require expensive licenses. Learning Linux administration opens the door to running your own infrastructure. You can host websites, set up email, run databases—all on servers you control.

The command line is your friend. Once you're comfortable navigating, configuring, and troubleshooting Linux, you'll understand how software actually works at a fundamental level.

WHAT YOU'LL DO

Set up and configure web servers like nginx and Apache. Manage users, permissions, and security settings. Automate tasks with shell scripts and cron jobs. Handle backups and disaster recovery. Monitor system resources and troubleshoot performance issues.

Run your own mail server so Google can't read your emails. Host your own file sync so Dropbox doesn't own your data. Set up your own VPN so your ISP can't track you. Self-hosting is the point.

THE SKILLS

Command line fluency. Understanding of the Linux filesystem hierarchy. User and group management. Package management. Service configuration (systemd). Networking basics. Shell scripting. Security hardening.

You'll learn to think like an administrator: anticipate failures, automate solutions, document everything. These habits serve you in every technical role.
